# 🚀 IdeaForge Live

### *Where startup ideas feel alive*

IdeaForge Live is an interactive web platform where students can share startup ideas, collaborate with others, and explore innovative concepts in real time.
The platform combines **live interaction**, **AI-powered idea generation**, and **modern UI themes** to create an engaging innovation ecosystem.

---

# 🌟 Features

## 🧠 Idea Sharing

* Submit startup ideas easily
* Add description, category, and tags
* Ideas appear instantly in the live feed
* Real-time updates without refresh

## 🤖 AI Idea Generator

* Generate startup ideas instantly
* Helps users brainstorm new concepts
* Automatically adds generated ideas to the feed

## 📊 Live Idea Feed

* View ideas posted by users
* Popularity tracking with visual bars
* See trending and active ideas
* Interactive collaboration system

## 💬 Collaboration Tools

* Vote on ideas
* Comment on ideas
* Reply to discussions
* Save favorite ideas

## 🔍 Search & Filter

* Search ideas by keywords
* Filter ideas by category:

  * Tech
  * Health
  * Social
  * Climate
* Sort by popularity or trending

## 🌗 Light & Dark Theme

* Toggle between light and dark modes
* Improved readability
* Modern UI experience

## ⚡ Smart Features

* Live idea updates
* Drag-to-reorder cards
* Local data persistence
* Trending ideas tracking
* Tag-based navigation

# 🧪 How to Run the Project

Follow these steps to run the project locally:

### Step 1 — Download the Project

Clone or download the repository:

```bash
git clone https://github.com/your-username/IdeaForge-Live.git
```

### Step 2 — Open Project Folder

Open the folder in **VS Code** or any code editor.

### Step 3 — Run the Website

Simply open:

```
startup.html
```

in your browser.
